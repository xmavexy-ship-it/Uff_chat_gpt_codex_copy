# Maweplayer

Красивый MP3-плеер для Android на Jetpack Compose.

## Что есть в приложении
- Градиентный интерфейс и карточка текущего трека.
- Кнопки `Previous / Play-Pause / Next`.
- Слайдер прогресса с перемоткой.
- Плейлист с подсветкой активного трека.
- Потоковое воспроизведение MP3 через Media3 ExoPlayer.

## Быстрый старт
1. Откройте проект в Android Studio (JDK 17 или 21).
2. Выполните синхронизацию Gradle.
3. Запустите приложение на эмуляторе/устройстве Android.

> Название плеера: **Maweplayer**.
